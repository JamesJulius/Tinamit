Ejemplo muy sencillo
--------------------
Este es un :ref:`ejemplo <ejemplo_senc>` con un modelo biofísico irrealista pero escrito únicamente en Python,
así que los puedes correr sin descargar nada. Ideal para hacer una prueba de Tinamït.

Salinidad de suelos (con SAHYSMOD)
----------------------------------
Este :ref:`ejemplo <ejemplo_sal>`, un poco más realístico que el anterior, muestra cómo se pueden emplear y
automatizar combinaciones complejas de varias políticas en un contexto de análisis de cambios climáticos,
generando mapas de impactos de políticas para cada combinación de políticas y escenario climático.

Seguridad alimentaria (con DSSAT)
---------------------------------
Este :ref:`ejemplo <ejemplo_segali>` analiza causas de inseguridad alimentaria en Guatemala a distintas escalas.
Todavía en trabajo.
